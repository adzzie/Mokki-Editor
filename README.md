Mokki-Editor
============

Mokki Editor is a lightweight HTML5 WYSIWYG / HTML Rich Text Editor. Some feature are included :

	- HTML live preview
	- Embed video iframe
	- Insert image location
	- Light Weight
	
Instalation
-----------
	
	Mokki.editor('.damnEditor');

Requirements
------------

	- jquery-1.11.1.min.js
	- font-awesome-4.1.0 or Higher
